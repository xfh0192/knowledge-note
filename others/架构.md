# 中台【开发中台专题】

## 什么是中台
- 减少重复劳动
- 赋能
- 平台化

## 中台的划分
- 前台与后台之间
- 技术中台
- 业务中台
- 组织中台

## 代码概念
- 元数据(metadata) 描述数据的数据。如http起始行、头部，是描述body的数据
- 高阶函数 处理函数的函数、返回函数的函数
  - 固化处理逻辑
  - 留空处理逻辑
- proxy 代理。proxy server、ES6 Proxy feature
  - 代替用户做某件事
  - 无感知的优化
